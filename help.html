Website Emas - Help

Registrasi

Registrasi diperlukan agar anda sebagai pengguna program dikenali oleh sistem sehingga ketika anda memutuskan untuk berlangganan program ini sistem akan mencatat nama dan jenis langganan yang anda pilih. Ada 3 (tiga) jenis langganan, yakni Langganan bulanan, Langganan tahunan dan Langganan satu kali beli untuk pemakaian selamanya.

Hitung Kadar Emas

Program ini adalah untuk menghitung kadar emas berdasarkan berat kering, berat basah dan kondisi barang dilihat dari warna. Caranya masukkan berat kering (berat di luar air) dan berat basah (berat di dalam air), pilih kondisi warna lalu klik tombol Hitung.

Struk Menggunakan Berat Basah

Program ini disajikan untuk membuat struk jual-beli emas dengan cara memasukkan berat kering, berat basah dan kondisi warna untuk menghitung kadarnya. Caranya sama seperti pada menu Hitung Kadar Emas tetapi dibuat untuk satu transaksi dengan satu penjual yang terdiri dari banyak item (jenis emas).

Tiap item akan tersimpan dalam daftar sementara yang muncul di bagian bawah halaman ketika anda klik tombol Simpan. Prosedur diulang hingga semua item tersimpan dalam daftar sementara. Jika semua item sudah masuk dalam daftar sementara anda bisa mencetak daftar sementara tersebut dengan cara klik tombol Cetak. Jika hape anda terhubung dengan printer melalui koneksi bluetooth maka hasil cetakan akan keluar untuk diserahkan kepada penjual. Tetapi jika hape anda tidak terhubung dengan printer maka anda bisa pilih Simpan ke PDF, selanjutnya system akan menyimpan struk tersebut ke dalam folder Download dengan nama file struk.pdf. File pdf itu lalu anda kirim kepada penjual.

Struk Menggunakan Kode Barang

Program ini fungsinya sama seperti fungsi program Struk Menggunakan Berat Basah yakni untuk membuat struk jual-beli emas, tetapi di sini kadar emas sesungguhnya sudah diketahui berdasarkan kode barang yang tertulis pada kuitansi yang dikeluarkan oleh toko emas di mana penjual itu membeli emas tersebut. Dengan demikian harga bisa ditetapkan berdasarkan kode barang tersebut.

Prosedur memasukkan item ke dalam daftar sementara hingga proses mencetak sama seperti pada program Struk Menggunakan Berat Basah sebagaimana dijelaskan di atas.
